# Homebrew Tap for Yoke

**Domain intelligence in one pass.**

## Install

```bash
brew install yokedotlol/tap/yoke
```

## Update

```bash
brew upgrade yoke
```

## About

This tap is automatically updated by [GoReleaser](https://goreleaser.com) on each [Yoke release](https://github.com/yokedotlol/yoke/releases).

For more information, visit [yoke.lol](https://yoke.lol) or the [main repository](https://github.com/yokedotlol/yoke).
